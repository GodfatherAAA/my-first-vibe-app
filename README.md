# My First Vibe App

Ứng dụng di động React Native (Expo) với màn hình Đăng nhập.

## Cấu trúc thư mục

```
├── App.tsx                 # Điểm vào ứng dụng
├── app.json                # Cấu hình Expo
├── assets/                 # Icon, splash (thay bằng file thật khi publish)
└── src/
    ├── components/         # UI tái sử dụng (input, nút, logo…)
    ├── screens/            # Màn hình (LoginScreen…)
    └── theme/              # Màu sắc, khoảng cách
```

## Chạy ứng dụng

```bash
npm install
npm start
```

Quét mã QR bằng **Expo Go** trên điện thoại, hoặc nhấn `a` (Android) / `i` (iOS simulator).
